## 🚀 كيفية الاستخدام

### البدء السريع

1. **تثبيت المكتبات:**
```bash
npm install
```

2. **تشغيل المشروع:**
```bash
npm run dev
```

3. **بناء للإنتاج:**
```bash
npm run build
```

### الخطوات الأساسية

#### 1. **إعداد مفتاح API:**
- اذهب إلى الإعدادات
- أدخل مفتاح Gemini API
- احصل عليه من: https://aistudio.google.com/app/apikey

#### 2. **إنشاء مشروع:**
- انقر على "مشروع جديد"
- أدخل اسم المشروع ووصفه
- اختر أيقونة ولون

#### 3. **إضافة البيانات:**
- رفع ملف (Excel, CSV)
- لصق بيانات مباشرة
- استيراد من قاعدة بيانات

#### 4. **التحليل:**
- انقر على "تحليل بالذكاء الاصطناعي"
- انتظر النتائج (5-15 ثانية)

#### 5. **عرض التقرير الشامل:**
- انقر على "📄 عرض التقرير الشامل"
- تصدير كـ PDF, Word, HTML
- طباعة مباشرة

#### 6. **استكشاف النتائج:**
- تبويب التحليل - الرؤى والتوصيات
- تبويب التصورات - المخططات البيانية
- تبويب الدردشة - محادثة مع البيانات

## 🎓 أمثلة الاستخدام

### مثال 1: تحليل المبيعات
1. ارفع ملف Excel يحتوي على بيانات المبيعات
2. انقر على "تحليل"
3. احصل على:
   - أفضل المنتجات مبيعاً
   - الاتجاهات الشهرية
   - التنبؤات المستقبلية
   - توصيات لزيادة المبيعات

### مثال 2: تقرير للإدارة
1. حلل البيانات
2. انقر على "عرض التقرير الشامل"
3. صدّر كـ PDF
4. أرسل للإدارة

### مثال 3: عرض تقديمي للعميل
1. حلل بيانات العميل
2. انقر على "تصدير" > "PowerPoint"
3. احصل على عرض تقديمي احترافي جاهز

### مثال 4: تحليل العملاء
1. استورد بيانات العملاء من API
2. اطلب من AI: "ما هي شرائح العملاء الرئيسية؟"
3. احصل على:
   - تصنيف العملاء
   - أنماط الشراء
   - فرص النمو
