## 🚀 كيفية الاستخدام

### البدء السريع

1. **تثبيت المكتبات:**
```bash
npm install
```

2. **تشغيل المشروع:**
```bash
npm run dev
```

3. **بناء للإنتاج:**
```bash
npm run build
```

### الخطوات الأساسية

1. **إعداد مفتاح API:**
   - اذهب إلى الإعدادات
   - أدخل مفتاح Gemini API
   - احصل عليه من: https://aistudio.google.com/app/apikey

2. **إنشاء مشروع:**
   - انقر على "مشروع جديد"
   - أدخل اسم المشروع ووصفه
   - اختر أيقونة ولون

3. **إضافة البيانات:**
   - رفع ملف (Excel, CSV)
   - لصق بيانات مباشرة
   - استيراد من قاعدة بيانات

4. **التحليل:**
   - انقر على "تحليل بالذكاء الاصطناعي"
   - انتظر النتائج (5-15 ثانية)

5. **استكشاف النتائج:**
   - تبويب التحليل - الرؤى والتوصيات
   - تبويب التصورات - المخططات البيانية
   - تبويب الدردشة - محادثة مع البيانات

6. **التصدير:**
   - انقر على "تصدير"
   - اختر الصيغة المطلوبة
   - حمل الملف

## 🎓 أمثلة الاستخدام

### مثال 1: تحليل المبيعات
1. ارفع ملف Excel يحتوي على بيانات المبيعات
2. انقر على "تحليل"
3. احصل على:

   - الاتجاهات الشهرية
   - التنبؤات المستقبلية

3. احصل على:
   - تصنيف العملاء
   - أنماط الشراء
