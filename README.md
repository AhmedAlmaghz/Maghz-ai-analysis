## 📋 نظرة عامة

**محلل البيانات الذكي** هو تطبيق ويب احترافي متكامل لتحليل البيانات باستخدام الذكاء الاصطناعي. يوفر منصة شاملة لرفع البيانات من مصادر متعددة (ملفات Excel، CSV، قواعد البيانات)، وتحليلها تلقائياً باستخدام Gemini AI، واستخراج الرؤى والتنبؤات والتوصيات العملية.

### 🎯 المشكلة التي يحلها

في عالم البيانات الضخمة، يحتاج المحللون وأصحاب الأعمال إلى:
- ✅ تحليل سريع ودقيق للبيانات دون الحاجة لخبرة برمجية
- ✅ رؤى ذكية وتنبؤات مبنية على أنماط البيانات الفعلية
- ✅ تصورات بيانية احترافية وتفاعلية
- ✅ توصيات عملية قابلة للتنفيذ
- ✅ واجهة عربية سهلة الاستخدام

## ✨ المميزات

### 📊 **تحليل البيانات المتقدم**
- 📁 رفع ملفات Excel (XLSX, XLS) و CSV
- 📋 لصق البيانات مباشرة (CSV/TSV)
- 🗄️ الاتصال بقواعد البيانات (MySQL, PostgreSQL, REST API, GraphQL, MongoDB)
- 🔍 تحليل تلقائي لأنواع الأعمدة والإحصائيات
- 📈 تصورات بيانية تفاعلية (Bar, Line, Pie, Area Charts)

### 🤖 **الذكاء الاصطناعي (Gemini 3.1 Flash Lite)**
- 💡 استخراج الرؤى الرئيسية تلقائياً
- 📊 توليد مؤشرات الأداء (KPIs) الذكية
- 📈 تحليل الاتجاهات والأنماط
- ⚠️ اكتشاف الشذوذ والقيم غير العادية
- 🎯 التنبؤات المستقبلية المبنية على البيانات
- ✅ توصيات عملية قابلة للتنفيذ

### 💬 **الدردشة الذكية**
- 🗨️ محادثة طبيعية مع البيانات
- 📊 توليد تصورات بيانية ديناميكية حسب الطلب
- 💡 اقتراحات ذكية للمتابعة بعد كل رد
- 💾 حفظ تلقائي لسجل المحادثة
- 🎨 عرض الرسوم البيانية داخل المحادثة

### 📁 **إدارة المشاريع**
- 🗂️ إنشاء مشاريع متعددة
- 🎨 تخصيص الأيقونات والألوان
- 🏷️ نظام الوسوم (Tags) للتصنيف
- ⭐ المفضلة للوصول السريع
- 🔍 البحث والتصفية المتقدم

### 🔌 **الاتصالات المتقدمة**

## 🚀 التثبيت

### المتطلبات الأساسية
- Node.js 18+ 
- npm أو yarn أو pnpm

### خطوات التثبيت

```bash
# 1. استنساخ المشروع
git clone https://github.com/yourusername/ai-data-analyzer.git
cd ai-data-analyzer

# 2. تثبيت المكتبات
npm install

# 3. تشغيل المشروع في وضع التطوير
npm run dev

# 4. بناء المشروع للإنتاج
npm run build

# 5. معاينة البناء
npm run preview
```

### 🔑 الحصول على مفتاح Gemini API

1. اذهب إلى [Google AI Studio](https://aistudio.google.com/app/apikey)
2. سجّل دخولك بحساب Google
3. اضغط على "Create API Key"
4. انسخ المفتاح واستخدمه في التطبيق

> 💡 **ملاحظة:** المفتاح يُحفظ محلياً في المتصفح فقط ولا يتم إرساله لأي خادم خارجي

## 📖 الاستخدام

### 1️⃣ إنشاء مشروع جديد

1. من لوحة التحكم، اضغط على **"مشروع جديد"**
2. أدخل اسم المشروع والوصف
3. اختر أيقونة ولون
4. أضف وسوم (اختياري)
5. اضغط **"إنشاء المشروع"**

### 2️⃣ إضافة البيانات

#### **الطريقة 1: رفع ملف**
- اضغط على **"رفع ملف"**
- اختر ملف Excel أو CSV
- انتظر حتى يتم التحميل

#### **الطريقة 2: لصق بيانات**
- اختر تبويب **"لصق بيانات"**
- الصق البيانات بصيغة CSV/TSV
- اضغط **"تحليل البيانات"**

#### **الطريقة 3: قاعدة بيانات**
- اختر تبويب **"قاعدة بيانات"**
- اختر اتصالاً محفوظاً أو أنشئ اتصالاً جديداً
- اضغط على الاتصال لاستيراد البيانات

### 3️⃣ التحليل بالذكاء الاصطناعي

1. تأكد من إضافة مفتاح API في **الإعدادات**
2. اضغط على **"تحليل بالذكاء الاصطناعي"**
3. انتظر حتى يكتمل التحليل (5-15 ثانية)
4. استعرض النتائج:
   - 📊 الملخص التنفيذي
   - 💡 الرؤى الرئيسية
   - 📈 الاتجاهات
   - ⚠️ الشذوذ
   - 🎯 التنبؤات
   - ✅ التوصيات

### 4️⃣ الدردشة الذكية

1. انتقل إلى تبويب **"الدردشة"**
2. اكتب سؤالك أو استخدم الاقتراحات الجاهزة
3. اضغط **Enter** أو زر الإرسال
4. الذكاء الاصطناعي سيجيب ويقترح:
   - 📊 تصورات بيانية (إذا طُلبت)
   - 💡 اقتراحات للمتابعة
5. اضغط على أي اقتراح لمتابعة المحادثة

### 5️⃣ التصورات البيانية

1. انتقل إلى تبويب **"التصورات"**
2. اختر العمود الرقمي والفئوي
3. استعرض الرسوم التلقائية:
   - 📊 Bar Chart (التوزيع)
   - 🥧 Pie Chart (النسب المئوية)
   - 📈 Line/Area Chart (الاتجاه الزمني)
   - 🎯 Forecast (التنبؤ المستقبلي)

### 6️⃣ التصدير

1. اضغط على زر **"تصدير"**
2. اختر الصيغة:
   - 📄 **JSON** - المشروع كاملاً
   - 📊 **CSV** - البيانات فقط
   - 💬 **TXT** - سجل الدردشة
   - 🖼️ **PDF** - تقرير احترافي
